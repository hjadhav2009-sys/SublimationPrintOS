# Product Brief

SublimationPrintOS is an offline Windows desktop production OS for sublimation businesses.

## Core Workflow

Raw image/design -> AI upscale/QC -> Design Store -> Design Studio -> Order Basket -> Auto Nesting -> Export Print Sheet -> Hardware/Production later.

## Current Status

The app is Phase 0 scaffold only. It currently provides the desktop foundation, UI shell, placeholder pages, and basic Tauri command wiring.

## Explicit Exclusions

For now, SublimationPrintOS does not include inventory, CRM, shipping, returns/RTO, accounting, marketplace system, or warehouse system features.
